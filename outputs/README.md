Simulated atmospheric microplastics
===================================

This directory contains outputs of the [GEOS-Chem](https://geoschem.github.io/) simulations of atmospheric microplastic emissions, transport, and deposition described in [TODO PAPER REF].


Contents
--------

### alt

Outputs of the alternate simulation that uses the configuration described in Fu et al. (2023)

### main

Outputs of the main simulation that uses a power law particle size distribution for atmospheric microplastic emissions
