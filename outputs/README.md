Simulation outputs
==================

This directory contains the raw simulation outputs.

These outputs are **not** constrained to match observations. The constrained outputs are available at https://doi.org/10.5281/zenodo.20922804.


Contents
--------

### [alt/](alt/)

Raw outputs of the *alternate* simulation using the configuration described in Fu et al. (2023).

### [main/](main/)

Raw outputs of the *main* simulation using a power law particle size distribution for atmospheric microplastic emissions.
