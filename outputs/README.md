Simulated atmospheric microplastics
===================================

This directory contains the raw outputs of the atmospheric microplastic cycling simulations described in "Reduced global atmospheric microplastic emissions from size-harmonized observations" (Hough et al., 2026).

These outputs are **not** constrained to match observations. The constrained outputs are available at [doi:10.5281/zenodo.20922804](https://doi.org/10.5281/zenodo.20922804).
