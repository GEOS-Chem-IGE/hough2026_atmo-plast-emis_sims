Simulation configurations
=========================

This directory contains configurations for the GEOS-Chem simulations of atmospheric microplastic cycling described in "Reduced global atmospheric microplastic emissions from size-harmonized observations" (Hough et al., 2026).

The outputs and restart files from these simulations are written to the [outputs/](/outputs/) directory via symlinks.


Contents
--------

### [alt/](alt/)

*Alternate* simulation using the configuration described by Fu et al. (2023). See the *alt* [README](alt/README.md) for details.

### [main/](main/)

*Main* simulation using a power law particle size distribution for atmospheric microplastic emissions. See the *main* [README](main/README.md) for details.

### [setup/](setup/)

Configuration and scripts to build the microplastics-enabled GEOS-Chem model and download input data.
